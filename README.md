## Name  
arduino_unitylight  

## Overview
## Description   
unity上のuguiにボタンを配置。そのボタンをクリックドリブンで、arduinoにつないである数種類の電球をLチカさせる。  
-こちらはunity(C#)側のコード。

## VS. 

## Requirement
-arduino  
-ワイヤー  
-電球  
-usbケーブル  
-unity画面上で必要なUGUIとarduinoを操作するためのunity(C#)側のコード。

## Usage
1.arduinoをシリアル通信で接続  
2.arduinoにLチカ用のセッティング  
3.Unity上にuguiのボタンを設定。C#で動作をかく。
